# Student Management App JEE

Mini application JEE de gestion d'etudiants developpee dans le cadre de l'atelier JEE a l'Ecole Superieure de Technologie d'Essaouira (Universite Cadi Ayyad), filiere ISIL.

## Technologies utilisees

* Servlet - Couche controleur
* JSP + JSTL - Couche presentation
* DAO + JDBC - Acces aux donnees
* MySQL - Base de donnees
* Docker + Docker Compose - Conteneurisation
* Maven - Gestion de projet

## Fonctionnalites

* Ajouter un etudiant
* Modifier un etudiant
* Supprimer un etudiant
* Afficher la liste des etudiants

## Lancement du projet

```bash
git clone https://github.com/AMALOUZ/student-management-jee.git
cd student-management-jee
docker-compose up
```

## Architecture

* Controller : Servlets
* View : JSP / JSTL
* Model : DAO + JDBC
* Base de donnees : MySQL
